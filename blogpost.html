HTML Dan JavaScript

Posted by Admin on 22/07/2022

HTML

HTML merupakan suatu bahasa yang digunakan untuk mendeskripsikan web pages.

1. Sintaks HTML

HTML elemen diawali dengan awal tag < .. > dan diakhiri dengan akhir tag < / .. >. Isi elemen adalah semua yang berada diantara awal tag dan akhir. Skema dasar HTML, yaitu :
< html >
< head >
< title > Judul Dokumen< /title >
< /head >
< body >Isi Dokumen< /body >
< /html >

2. Tag Dan Properti HTML

- Tag Paragraf = untuk menandai suatu paragraf
< p >paragraf< /p >

- Tag Atribut (bold, italic, underline)
< b >cetak tebal< /b >
< i >cetak miring< /i >
< u >garis bawah< /u >

- Tag Ganti Baris (Break Row)
< br >< /br >

- Tag Enumerasi = untuk menandai suatu item daftar
< li >item< /li >

- Tag Gambar = untuk menampilkan sebuah file gambar
< img src="namafilegambar" alt="" >

- Tag Link
< a href="link">< /a >

- Tag Tabel = untuk mendefinisikan sebuah tabel
< table >< /table >

3. Frame Dan Form

- Frame berfungsi untuk membagi beberapa tampilan yyang berbeda dalam satu jendela browser.
Sintaks Tag Frame = < frame >< /frame >

- Form digunakan untuk memperoleh data-data user baik nama, alamat dan data lainnya untuk mendaftar pada service yang disediakan.
Sintaks Form = < Form >< /Form >

JavaScript

1. Sintaks JavaScript

Peletakan JavaScript di HTML dengan tag < scrip t languange="JavaScript">< /scrip t >

2. Variabel JavaScript

Variabel adalah suatu obyek yang berisi data data, yang mana dapat dimodifikasi selama pengeksekusian. Mendeklarasikan variabel eksplisit dengan menuliskan kata kunci var kemudian diikuti dengan nama variabel dan nilai dari variabel, mendeklarasikan variabel implisit dengan menuliskan secara langsung nama dari variabel dan diikuti nilai dari variabel.

3. Tipe Data JavaScript

- Integral (Bilangan Bulat)
- Float (Bilangan desimal)
- String (Kumpulan dari karakter)

4. Operator JavaScript

- Modulus (%)
- Penaikan (++)
- Penurunan (--)
